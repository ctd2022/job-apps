üéØ Command Line Reference

Core Parameters

  • --cv - Path to your CV file (required)
  • --job - Path to job description (required)
  • --company - Company name for personalized output & ATS filtering
  • --profile - Optional web profile content
  • --questions - Optional application questions file

Backend Options

  • --backend ollama - Use local Ollama (default)
    • --ollama-model MODEL - Specify model (default: llama3.1:8b)
  • --backend llamacpp - Use Llama.cpp server
    • --llamacpp-url URL - Server URL (default: http://localhost:8080)
    • --llamacpp-model NAME - Model name for metadata
  • --backend gemini - Use Google Gemini API
    • --gemini-key KEY - API key (or set GEMINI_API_KEY env var)
    • --gemini-model MODEL - Model name (default: gemini-1.5-pro)
    • --gemini-rpm N - Requests per minute (default: 10)

Feature Flags

  • --no-ats - Disable ATS optimization (for small companies/referrals)
